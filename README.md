# Little Lemon Table Booking App

## Project Overview
This is a React web application that allows users to book a table at the Little Lemon restaurant. The app includes form validation, accessibility features, responsive design, and unit testing.

---

## Features
- Table booking form
- Input validation
- Accessible labels and semantic HTML
- Responsive UI
- Unit test for booking form

---

## Technologies Used
- React
- JavaScript
- CSS
- React Testing Library

---

## Installation & Setup

Clone the repository:

git clone https://github.com/archit-react/little-lemon-capstone.git

Navigate into project:

cd little-lemon-capstone

Install dependencies:

npm install

Run the app:

npm start

---

## Running Tests

npm test
